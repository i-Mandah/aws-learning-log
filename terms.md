# Terms I learned, in plain language

- Server: a computer that runs all the time and is reachable over the internet.
- Cloud server: a server you rent from Amazon instead of owning.
- SSH: a secure remote door into another machine, opened with a key.
- Image: a sealed box holding an app and everything it needs to run.
- Container: an image opened and running.
- Registry: the warehouse images are pulled from, like Docker Hub.
- Daemon: the Docker engine running in the background.
- vole: a small rat (test word)
